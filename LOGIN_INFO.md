## 일반 사용자 계정

### 회원가입하기 (3가지 방법)

#### 방법 1: 이메일 회원가입
1. "📝 회원가입" 버튼 클릭
2. 이름, 이메일, 학년반호, 비밀번호 입력
3. 가입하기 버튼 클릭

#### 방법 2: Google 계정으로 회원가입
1. "📝 회원가입" 버튼 클릭
2. "Google로 회원가입" 버튼 클릭
3. Google 계정으로 로그인
4. 자동으로 회원가입 및 로그인 완료

### 로그인하기 (2가지 방법)

#### 방법 1: 이메일 로그인
1. "🔐 로그인" 버튼 클릭
2. 이메일과 비밀번호 입력
3. 로그인 버튼 클릭

#### 방법 2: Google 계정으로 로그인
1. "🔐 로그인" 버튼 클릭
2. "Google로 로그인" 버튼 클릭
3. Google 계정으로 로그인
4. 자동으로 로그인 또는 회원가입 완료

---

## Google 로그인 설정 (선택사항)

Google 로그인을 활성화하려면:

1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성
3. OAuth 2.0 클라이언트 ID 생성
4. `index.html`에서 `YOUR_GOOGLE_CLIENT_ID`를 실제 ID로 변경

```javascript
client_id: 'YOUR_GOOGLE_CLIENT_ID.apps.googleusercontent.com'
```

## 문제 해결

### Q: 로그인이 안 된다
**A:** 
- 이메일이 `abc@admin.com`인지 확인 (관리자)
- 비밀번호가 `1234`인지 확인
- 일반 사용자는 회원가입한 이메일과 비밀번호 입력

### Q: Google 로그인 버튼이 안 보인다
**A:**
- Google 로그인은 선택사항입니다
- YOUR_GOOGLE_CLIENT_ID를 실제 ID로 변경하면 활성화됩니다
- 이메일/비밀번호 로그인으로도 가능합니다

### Q: 데이터가 안 저장된다
**A:**
- 브라우저의 로컬 스토리지를 사용합니다
- 개발자 도구 → Application → Local Storage 확인

## 🎯 빠른 테스트

### 관리자로 로그인
```
이메일: abc@admin.com
비밀번호: 1234
```

### 공지사항 작성
1. 관리자로 로그인
2. "📢 공지사항" 클릭
3. "새 공지 작성" 버튼 클릭
4. 제목, 내용 입력
5. 작성 버튼 클릭

### 일반 사용자 회원가입 및 로그인
1. "📝 회원가입" 클릭
2. 정보 입력 또는 Google 계정 선택
3. 가입하기 클릭
4. 로그인

### Google 계정 사용
1. "📝 회원가입" 또는 "🔐 로그인" 클릭
2. Google 버튼 클릭
3. Google 계정으로 인증
4. 자동으로 회원가입 및 로그인 완료
